How to Use

After registering and obtaining your API key, you can make requests to our API endpoints:

GET /api/weather/{location}

Example response:

{
  "result": "SUCCESS",
  "message": "Weather data retrieved successfully",
  "data": {
    "location": "New York",
    "currentTemp": 22.5,
    "conditions": "Clear",
    "humidity": 65.2,
    "forecast": [...]
  }
}

Features

  • Real-time weather data
  • 5-day forecast predictions
  • Temperature, humidity, wind speed, and more
  • Support for cities worldwide
  • JWT-based authentication
  • Rate limiting to prevent abuse
